 # java-base-type-utils 提供java基础和部分扩展类型的链式调用，丰富扩展函数

给java的基础类型提供了扩展函数

## 使用方法
定义想要的变量。然后直接链式调用
eg：
```
String a;
a.customIsNull();
```
