# 是否为空-customIsNull()

### 入参

无

### 返回值

```
Boolean result
```

如果不为空返回false

# 判断正负数-customIsSign()

## customIsSign()

### 入参

无

### 返回值

```
Integer result
```

如果 source大于0则返回1
如果 source等于0则返回0
如果 source小于0则返回-1

### 实现功能
判断一个Double是正数还是负数还是0

### 可能的异常
如果有任意为空的数则直接throw
